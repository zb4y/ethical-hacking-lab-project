# Pentesting Methodologies Lab Report

This lab is part of my ethical hacking course.

## Objective
Compare different penetration testing methodologies and understand their use.

## Methodologies
- OSSTMM
- PTES
- OWASP WSTG
- MITRE ATT&CK

## Key Points

### OSSTMM
- Structured security testing methodology
- Focuses on measurable results
- Still useful but considered older

### PTES
- Full penetration testing process
- Includes planning, testing, and reporting

### OWASP WSTG
- Focused on web application security
- Useful for finding web vulnerabilities

### MITRE ATT&CK
- Based on real-world attacker behavior
- Helps understand attack techniques

## What I did in the lab
- Researched each methodology
- Answered lab questions
- Compared their strengths and use cases

## What I learned
Each methodology has a different purpose.  
A penetration tester should choose the right one depending on the situation.
